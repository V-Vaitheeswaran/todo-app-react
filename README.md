# 📝 Todo App (React)

A clean and modern Todo application built using React.

## 🚀 Features
- ➕ Add tasks
- ❌ Delete tasks
- ✏️ Edit tasks
- ✅ Mark as completed

## 🛠️ Tech Stack
- React (useState)
- CSS
